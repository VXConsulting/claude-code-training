# Claude Code Training

> Cours complet pour maîtriser Claude Code : de l'utilisation basique à la création de plugins avancés.

## Objectifs

Ce tutoriel vous apprendra à :

- Utiliser Claude Code efficacement au quotidien
- Comprendre et exploiter tous les outils natifs
- Créer des hooks pour automatiser et sécuriser vos workflows
- Développer des skills réutilisables
- Configurer des agents spécialisés
- Intégrer des serveurs MCP

## Prérequis

- Claude Code installé (`npm install -g @anthropic-ai/claude-code`)
- Connaissances de base en ligne de commande
- Un éditeur de code (VS Code recommandé)

1. Clonez ce repository
2. Suivez les modules dans l'ordre
3. Complétez les exercices pratiques de chaque module
4. Le projet final consolide tous les apprentissages
